Sample Simulation Data - README
===============================

Description:
------------
This folder contains small example CSV outputs generated by the simulation code.

The full research runs used much larger parameter grids. This repository includes only small
sample files so the project remains easy to inspect and run.

CSV Format:
-----------
Each output file has the following columns:

    delta,epsilon,final_alpha,final_alpha_prime,steps,rms

Column Explanation:
-------------------
- "delta" is the dimensionless driving frequency.
- "epsilon" is the dimensionless driving amplitude.
- "final_alpha" is the final angular position after the simulation.
- "final_alpha_prime" is the final angular velocity after the simulation.
- "steps" is the number of integration steps performed.
- "rms" is the RMS amplitude after transients.

Use of RMS:
-----------
The "rms" value is the main quantity used to build parameter maps of the system response.

Small RMS values correspond to damped or nearly pinned behavior. Non-zero stable RMS values
indicate persistent finite-amplitude oscillations.

Notes:
------
Large parameter-grid outputs are not included in this repository because they are generated
files and can be reproduced from the configuration files.

This folder is meant to show the output format and provide small example data for inspection.
